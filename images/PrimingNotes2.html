New Approach¶

Potential problem: Low-exposure targets are inherently more difficult¶

Solution: Can't say that certain targets were more difficult if only using a single target¶

Potential problem: Total exposure measure previously collapsed data from early and late in game into one averaged data point¶

Solution: Now exposure is calculated on an individual trial basis¶

Potential problem: Same trials occur less often than diff trials¶

Solution: If there is more variability in one of the line graphs (indicated by transparent colors), that doesn't undermine analysis. Can still look at difference between two regressions¶

  • Also this would allow more trials to be included, as we no longer need to cutoff with players contributing same number of trials to same and different.

Potential problem: Pro players would never be able to contribute to far right side of graph¶

Solution: Introduce cutoff where player needs to at least contribute to x number of exposures for inclusion in the data¶

Potential problem: Far right of the graph comes from levels of harder difficulty (i.e., faster conveyor belt speeds)¶

Solution: Construct another plot where x-axis represents total trials from player at that point in the game¶

  • This would more fully represent difference in priming benefit due to harder difficulty levels. Doing hierarchical analysis on difference between these two graphs with difference on the above graph would allow us to see if exposure explains priming benefit above and beyond level factors in the model.

New Approach Results¶

So I did all the above and one problem is that you cannot simply regress out total trial number because it is inextricably linked with exposure number -- changing the x-axis would merely stretch our the line plot and not be useful. So instead I substituted out the total trial number for the trial onset of that target for each individual player. Trial onset can inform us about trial difficulty (i.e., later trial onset should lead to smaller priming benefits overall if difficulty really were responsible for our effects and not exposure).

This is the current pipeline:

For item number x
   For participant y
       Find all trials containing item x for participant y
           First trial with this item is recorded as trial_onset
              Find all trials passing exclusionary criteria
                  If participant has seen this item >= 100 times (else disregard player)
                      Log following variables
                          If "same" (each item_num has separate cell)
                              RT, exposure_num, par_num, trial_onset
                          If "diff" (saved as separate matrices from "same")
                              RT, exposure_num, par_num, trial_onset

Would have to repeat this all for accuracy instead of RT as well.

Note that number of trials included for "same" and "diff" conditions are no longer equal. This is because we are constructing a regression independently for both "diff" and "same" (and then looking at the difference in their slopes). I.e., we would predict that the regression slopes would be significantly different for "diff" and "same". Also note that no player can bias an exposure's RT more than any other player, as it is impossible to contribute more than one data point for the same exposure_number for any given target.

Now we need to figure out what to do with all of this... It seems like trial onset isn't really changing the sizes of these effects much, and 13/15 targets showed priming benefit decrease over time.

I decided to replot all of the graphs given that the x-axis is now individual player's trial onset for a particular target. We should theoretically see a weaker effect now that we are graphing priming benefit according to when it was introduced to the player. If priming benefit is explained by trial difficulty though, we should predict that these graphs will show that later trial onset means less of an average priming difference. Again this is all the exact same data as above, just replotted according to individual's target trial onset.
